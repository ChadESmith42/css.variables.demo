Variables in CSS

Using CSS variables to create responsive websites.

What are CSS Variables

CSS variables allow the developer to set values that can be used over and over throughout the code, just like a variable in almost every other language.

For example, I set the following variables in the CSS stylesheet for this document:

:root {
   --base-font-size: 30px;
   --columns: 350px;
   --base-margin: 30px;
   --base-padding: 30px;
   --float-left: left;
   --float-right: right;
   --clear: none;
   --main-width: 60%;
   --aside-width: 30%;
   --main-left-margin: 30px;
   --aside-right-margin: 30px;
}

This code sets the font-size, column width, margin, and padding. If you notice, I've set variables for float and clear. This is really useful for modifying a page to "stack" on small screens, but use a main-aside style design on larger screeens.

I call the variable in my CSS like so:

 body {
    font-size: var(--base-font-size);
 }

Now, instead of rewriting all of the font-size, margin, padding, and column values in a media query, I can simply re-set the variable values, like so:

@media all and (max-width: 450px) {
  :root {
    --columns: 200px;
    --base-margin: 15px;
    --base-padding: 15px;
    --base-font-size: 20px;
    --float-left: none;
    --float-right: none;
    --clear: both;
    --main-width: 100%;
    --aside-width: 100%;
    --main-left-margin: 15px;
    --aside-right-margin: 0px;
   }
}

That's It!

For this site, I probably saved 50-100 lines of code by using variables instead of recoding all of the height, width, font, and positioning elements. Using variables also made it easier to handle the colors for the site. One method of using variables to handle colors would be to create a different varible for each color setting:

  --default-color: #black;
  --default-background-color: #white;
  --default-border-color: #ltgray;
  --hover-color: #white;
  --hover-background-color: #black;
  --hover-background-color: #ltblue;

In my example above, my variable names are really long, which doesn't seem like much of a time saver. However, if you have multiple color schemes, it might be a great way to keep everything organized. Originally, I had five different color sets I had sampled from the background image, Van Gogh's Irises In A Vase. In the end, I just didn't want to bother for such a simple project.

Go make something great with CSS variables!
